# Mission

一个零依赖的命令行 TODO 工具。打开终端就能记，不用等、不用联网、不用装任何东西。

## 要解决的问题

临时冒出来的想法、要做的事、需要跟进的点——这些东西不马上记下来就忘了。现有的笔记工具太重：要打开 App、等加载、甚至要登录。终端是开发者最常停留的地方，TODO 工具应该离手边不超过一个命令的距离。

## 价值主张

**比脑子可靠，比 App 轻。** 用最短路径把念头变成一条记录，不打断当前工作流。

## 目标用户

- **自己**（开发者），macOS 环境，终端是常驻应用
- 场景：写代码中途想起一件事 → `uv run python todo.py add "xxx"` → 继续干活
- 痛点：不想切窗口、不想等启动、不想引入一堆 node_modules 或依赖

## 利益相关方

| 谁 | 需要什么 |
|---|---|
| 自己（用户） | 随时记、随时查、随时划掉，零摩擦 |
| 自己（维护者） | 代码简单，标准库搞定，不用追版本 |

## 约束与边界

- **不做什么**：不同步、不协作、不提醒、不分类/标签、不设截止日期、不导出
- **不依赖第三方包**：只用 Python 标准库
- **不做交互式 TUI**：纯子命令 + 参数，一次调用一个动作
- **不做多用户 / 多设备**：数据存在用户家目录的 `~/.todo_cli/tasks.json` 里，全局唯一
- **只保证 macOS 可用**：不刻意做跨平台兼容

## 成功标准

> 在任意目录下，能在 2 秒内完成"记一条 TODO → 列出所有 TODO → 标记完成 → 删除"的完整闭环，且全程不产生任何网络请求或外部依赖安装。
